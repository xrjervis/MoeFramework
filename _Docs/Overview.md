# Overview

* * * * *

MoeFramework is a "Manager of Managers" based unity light framework designed for creating and managing a game more easily.

* MoeFramework
  * Core
    * GameManager
    * LevelManager
    * SaveManager
  * Events
    * EventManager
  * Singleton
    * MonoSingleton
    * Singleton
  * UI
    * GUIManager
    * UIBase
    * UIRootHandler
  * Utility
    * Utility
